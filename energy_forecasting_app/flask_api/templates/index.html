Energy Forecasting API

Welcome to the Energy Forecasting REST API. This API provides machine learning-powered predictions for energy consumption based on temporal features.

API Endpoints

GET /api/health

Check API health status and model loading status.

Response: { "status": "healthy", "model_loaded": true, "timestamp": "2024-01-01T12:00:00" }

GET /api/model/info

Get information about the loaded model including RMSE, features, and feature importance.

POST /api/predict/single

Predict energy consumption for a single datetime.

Request Body: { "datetime": "2024-06-15T14:00:00" } Response: { "success": true, "prediction": { "datetime": "2024-06-15T14:00:00", "prediction": 17234.56, "features": {...} } }

POST /api/predict/range

Predict energy consumption for a date range.

Request Body: { "start_date": "2024-06-15", "end_date": "2024-06-16", "frequency": "H" } Response: { "success": true, "predictions": [...], "summary": { "count": 24, "mean": 17234.56, "max": 19876.43, "min": 14567.89, "std": 1234.56 } }

POST /api/predict/chart

Generate predictions and return chart as base64 image.

Request Body: { "start_date": "2024-06-15", "end_date": "2024-06-16", "frequency": "H", "chart_type": "prediction" } Chart Types: "prediction", "hourly", "weekly"

GET /api/analysis/feature-importance

Get feature importance analysis with chart.

POST /api/analysis/patterns

Analyze energy consumption patterns with multiple charts and statistics.

Request Body: { "start_date": "2024-06-01", "end_date": "2024-06-07" }

Usage Examples

Python Example

import requests import json # Single prediction response = requests.post('http://localhost:5000/api/predict/single', json={'datetime': '2024-06-15T14:00:00'}) result = response.json() print(f"Predicted consumption: {result['prediction']['prediction']:.2f} MW") # Range prediction response = requests.post('http://localhost:5000/api/predict/range', json={ 'start_date': '2024-06-15', 'end_date': '2024-06-16', 'frequency': 'H' }) data = response.json() print(f"Generated {data['summary']['count']} predictions")

JavaScript Example

// Single prediction fetch('/api/predict/single', { method: 'POST', headers: {'Content-Type': 'application/json'}, body: JSON.stringify({datetime: '2024-06-15T14:00:00'}) }) .then(response => response.json()) .then(data => { console.log('Prediction:', data.prediction.prediction); });

Model Information

This API uses an XGBoost regression model trained on historical energy consumption data. The model uses time-based features including:

  • hour: Hour of the day (0-23)
  • dayofweek: Day of the week (0=Monday, 6=Sunday)
  • month: Month of the year (1-12)
  • quarter: Quarter of the year (1-4)
  • year: Year
  • dayofyear: Day of the year (1-365)

The model achieves an RMSE of approximately 1649 MW on the test set.
